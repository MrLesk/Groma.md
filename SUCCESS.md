# What Success Looks Like

This document is a non-normative product north star. [MANIFESTO.md](MANIFESTO.md) and
[ARCHITECTURE.md](ARCHITECTURE.md) remain authoritative.

Groma is successful when architecture stops being a picture someone once drew and
becomes the shared, living understanding that humans and agents use to change a
system.

**The map earns attention. Continuity earns trust.**

A beautiful diagram that goes stale is not success. A correct model that only experts
can use is not success. Groma succeeds only when it is obvious now and trustworthy
later.
