## Description

Deliver the first executable Groma vertical slice: initialize a local workspace, create and edit architectural intent through shared semantic operations, persist deterministic Markdown atomically, reload and query it through the same model, and ship the workflow as a compiled single-file Bun executable. The approved implementation stack is TypeScript and Bun; later local service and web work will use the Bun embedded server and Bun React bundler. Scanning, evidence reconciliation, disposable projection, dynamic plugin loading, plans, history, and web UI remain outside this milestone.
