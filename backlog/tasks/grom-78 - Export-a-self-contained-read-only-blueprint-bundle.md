## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
groma export writes a static bundle: the embedded web client with a bounded blueprint snapshot baked in, opening from a plain file with no server, no groma binary, and zero network requests. This is the expert-career-path dashboard model — build equals data snapshot plus read-only client, regenerated by CI on each commit and publishable as the team architecture dashboard. Read-only means read-only: every editing affordance stripped, and the bundle never becomes a mutation surface. Deterministic output for identical blueprint state keeps the bundle itself reviewable and CI-diffable.
<!-- SECTION:DESCRIPTION:END -->
